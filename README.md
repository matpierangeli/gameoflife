### install dependencies
```
sudo npm install
```
### run unit tests
```
mocha
```
### run app
```
node lib/app.js
```
or
```
node lib/app.js <sample-name>
```
where ```<sample-name>``` is a file name stored in 'resources' folder.  
eg. ```node lib/app.js gosper-glider-gun-extended.txt```

all samples are taken from:
